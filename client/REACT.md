리액트 시작하기
----------------

1. node.js 설치
1. node.js 및 npm 설치 확인
    > $ node -v  
      $ npm -v
1. create-react-app 설치
    > $ npm install -g create-react-app  
      $ create-react-app **./프로젝트명**

1. react 서버 실행
    > $ cd **./프로젝트명**  
      $ npm start  
      
1. *http://localhost:3000 접속*
